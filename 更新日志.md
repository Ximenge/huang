## v1.0.0 - 集成 ExoClick Popunder 弹窗广告 (2026-03-07 02:51:00)

### 功能

1. **ExoClick Popunder 弹窗广告集成**
   - 在网站所有页面集成 ExoClick 官方 Popunder 广告代码
   - 实现点击触发弹窗广告功能
   - 支持频率控制，避免过度打扰用户
   - 爬虫不显示广告，不影响 SEO

2. **创建简化版 Popunder 脚本**
   - 作为备用方案，提供自定义 Popunder 实现
   - 支持多种触发方式和配置参数

### 实现逻辑

#### Popunder 广告工作原理

1. **脚本加载**：页面加载时异步加载 ExoClick 官方代码
2. **配置读取**：从 `adConfig` 对象读取广告配置参数
3. **事件监听**：根据 `trigger_method` 设置点击事件监听
   - `trigger_method: 1` = 点击页面任意位置触发
   - `trigger_method: 2` = 点击特定 class 元素触发
   - `trigger_method: 3` = 点击链接时触发
4. **频率控制**：使用 Cookie 记录用户已看到的广告次数
   - Cookie 名称格式：`zone-cap-{idzone}`
   - 存储格式：`次数;时间戳`
5. **弹窗触发**：
   - 用户点击时，在新标签页打开当前页面链接
   - 在当前页面重定向到广告链接
6. **浏览器适配**：
   - Chrome：使用 `Ctrl+点击` 模拟方式
   - 移动端：使用兼容的弹窗方式
   - 其他浏览器：标准方式

#### 爬虫检测逻辑

```javascript
const botUserAgents = [
```

### 重要参数

#### ExoClick 广告配置 (来自 pp.md)

| 参数 | 值 | 说明 |
|------|-----|------|
| `ads_host` | `a.pemsrv.com` | 广告主机 |
| `syndication_host` | `s.pemsrv.com` | 同步主机 |
| `idzone` | `5866630` | 广告位 ID |
| `frequency_period` | `60` | 频率限制周期（分钟） |
| `frequency_count` | `2` | 周期内最多显示次数 |
| `trigger_method` | `1` | 触发方式（1=任意点击） |

#### 触发方式说明

| 值 | 触发方式 | 适用场景 |
|----|----------|----------|
| 1 | 任意点击 | 高收益，用户体验较差 |
| 2 | 特定 class | 精准控制，需要指定元素 |
| 3 | 点击链接 | 平衡收益和体验（推荐） |
| 4 | 非特定 class | 排除特定元素 |

## v1.0.3 - 留言板添加无限滚动分页功能 (2026-03-07 03:10:00)

### 功能

1. **留言板无限滚动分页**
   - 支持滚动到底部自动加载更多留言
   - 每页默认加载 10 条留言
   - 显示"加载更多..."和"没有更多留言了"状态提示
   - 提交留言后自动刷新并回到第一页

### 实现逻辑

#### 后端 API 分页

修改 `comments.ts` API：
- 支持 `?page=1&limit=10` 查询参数
- 返回分页信息：`{ comments, pagination: { page, limit, total, totalPages, hasMore } }`

#### 前端无限滚动

使用 `IntersectionObserver` API 实现无限滚动：
- 监听底部触发元素 `#load-more-trigger`
- 当元素进入视口时自动加载下一页
- 使用 `rootMargin: '100px'` 提前触发加载

### 重要参数

| 参数 | 值 | 说明 |
|------|-----|------|
| `limit` | 10 | 每页留言数量 |
| `rootMargin` | 100px | 提前触发加载的距离 |

#### API 响应格式

```typescript
interface ApiResponse {
  comments: Comment[];
  pagination: {
    page: number;       // 当前页码
    limit: number;      // 每页数量
    total: number;      // 总留言数
    totalPages: number; // 总页数
    hasMore: boolean;   // 是否有更多
  };
}
```
